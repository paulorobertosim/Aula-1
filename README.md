Um Technical Writer (redator técnico) é o profissional que transforma informações técnicas complexas, como códigos, APIs e funcionalidades de software, em documentações claras, 
acessíveis e úteis, como manuais, guias e FAQs

O Technical Writing (Escrita Técnica) foca em explicar produtos complexos através de manuais e documentações, priorizando a precisão técnica. Já o UX Writing (Escrita de Experiência do Usuário) desenha diálogos em interfaces digitais (botões, mensagens de erro) para tornar a navegação intuitiva, focando na jornada emocional do usuário
